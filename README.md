# Clarity
Chrome extension that highlights the most important sentences on any page, powered by a random forest model

phases:
1) Training Data
    Source 1 — Wikipedia lead sections
    Source 2 — News articles
2) Feature Extraction
3) Train the Model 
    The overfitting experiment 
4) Export to ONNX
    Format that lets Python-trained model run in JS
5) Build the Chrome Extension 
6) Load and Test
